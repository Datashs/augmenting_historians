# 1. Segment

> “La France du dix-neuvième siècle… m’offrait un double avantage…”

---

# 2. Toulmin analysis (logical structure)

## Key elements

- **Claim**  
  → The 19th-century French context is an appropriate framework for studying press corpora.

- **Grounds**  
  → familiarity with the terrain  
  → accessibility of digitized sources (Gallica, Retronews)  
  → importance of press in public debate  

- **Warrant**  
  → accessible and well-known sources justify the research choice  

- **Backing**  
  → historical practice and transformations of the discipline  

- **Qualifier**  
  → explicit recognition of limits  

- **Rebuttal**  
  → limits of individual work acknowledged  

## Observation

The structure is **complete and explicit**, including a rebuttal, which is relatively rare.

# Final note

This example shows how the pipeline operates across multiple layers:

- logical (Toulmin)  
- rhetorical (Perelman)  
- structural (RST)  
- synthetic (cross-analysis)  
- operational (recommendations)  

The system does not evaluate argumentation.

It provides:

> structured analytical perspectives on the same segment
